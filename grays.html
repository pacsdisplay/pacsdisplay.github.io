Gray Level Tool - Instructions
CONTROLS: - Left-click anywhere: Increase gray level by step size - Right-click anywhere: Decrease gray level by step size - Click the number: Type in a value (0-255) - Dropdown: Select step size (1, 5, or 15) - Grid toggle (⌗): Show/hide 3x3 grid overlay USAGE: - Gray levels range from 0 (black) to 255 (white) - Use this tool to evaluate display gray scale - Current gray level shown in top right (zero-padded) - Step size controls increment/decrement amount - Grid overlay helps assess uniformity across display
