CONTROLS: - Mouse Wheel Down / Right Arrow: Progress scanning box forward - Mouse Wheel Up / Left Arrow: Reverse scanning direction - Space: Toggle auto-play forward (stops at end or on scroll/arrow key) - Shift+Space: Toggle auto-play backward - Left-click / Up Arrow: Increase gray level to next value - Right-click / Down Arrow: Decrease gray level to previous value - Box Size Input: Set box dimensions (pixels, square) - Gray Dropdown: Select box gray level (15, 120, or 240) USAGE: - Box starts in the upper left corner (default 300x300 pixels) - Scroll down to move the box across the display, left to right then down to the next row, then right to left, and so on in serpentine fashion - Scrolling up reverses direction (including vertical movement) - Background is always black (gray level 0) - Use this tool to identify display non-uniformities URL PARAMETERS: - ?size=N - Set initial box size (e.g., ?size=300) - ?gray=N - Set initial gray level (e.g., ?gray=240) - Example: scan.html?size=150&gray=15 PURPOSE: - Scan the entire display surface systematically - Identify bad pixels and mura - Check uniformity at different gray levels
