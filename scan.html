pacsDisplay Uniformity Scan
CONTROLS: - Mouse Wheel Down: Progress scanning box forward - Mouse Wheel Up: Reverse scanning direction - Box Size Input: Set box dimensions (pixels, square) - Gray Dropdown: Select box gray level (15, 120, or 240) USAGE: - Box starts in the upper left corner (default 300x300 pixels) - Scroll down to move the box across the screen from left to right - When reaching the right edge, box moves down gradually - Then scans left, continuing in serpentine fashion - Scrolling up reverses direction (including vertical movement) - Background is always black (gray level 0) - Use this tool to identify display non-uniformities URL PARAMETERS: - ?size=N - Set initial box size (e.g., ?size=300) - ?gray=N - Set initial gray level (e.g., ?gray=240) - Example: scan.html?size=150&gray=15 PURPOSE: - Scan the entire display surface systematically - Identify bad pixels and mura - Check uniformity at different gray levels
